Treść pytania

We have Roman digits I, V, X, L, C.

We have to encode values 1-100. In the encoding, the section for every kind of digit in every possible position should be encoded separately. How many bits do we need?

Informacja zwrotna

A number has sections for numbers of Is, V, Xes, Ls, and Cs but in various positions in a sequence. 

The maximum number of Is is 3, as in III or VIII. 

The maximum number of Vs is 1, as in IV or VL. 

The maximum number of Xes is 3, as in XXX. 

The maximum number of Ls is 1, as in IL or LXXX. 

Xes can be before L or after L, if they are both.

Xes and/or Is can be before c, but not after

I can precede V, X, L, or C, and can be after V, X, or L.

And so on.

So the sections are Is, V, Is, Xes, Is, V, Is, L, Is, V, Is, Xes, Is, V, Is, C - totally 16 sections.

So we need bits 2+1+2+2+2+1+2+1+2+2+2+1+2+1=23

Poprawna odpowiedź to: 23

Informacja zwrotna
